# Endpoint Definitions

This project includes continuous acid-base outcomes and binary acidemia/metabolic acidosis endpoints.

## Continuous Outcomes

| Outcome | Description |
|---|---|
| `pH` | Umbilical artery blood gas pH |
| `pCO2` | Partial pressure of carbon dioxide |
| `BE` | Base Excess |
